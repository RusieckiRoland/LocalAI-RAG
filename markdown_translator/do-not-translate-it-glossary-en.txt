Do-Not-Translate (EN→PL) — IT Glossary
Practical list of English technical terms commonly left untranslated in Polish IT writing.

Notes
- This is a pragmatic industry list (docs, tickets, code reviews, UI strings).
- Leave these terms in English unless your style guide explicitly requires Polish equivalents.
- When in doubt: keep terms that appear in code/CLI/config, standards, acronyms, or product/feature names.
- This file is a human-oriented glossary; the translator runtime loads "never translate" terms from
  `templates.json` (see `never_translate_terms_files`), e.g. `never-translate-terms-safe.txt`.
